## 为什么工具很重要

好的工具能：

- 降低创作阻力
- 提高输出质量
- 节省时间精力
- 激发创作灵感

但记住：工具是手段，不是目的。不要为了工具而工具。

## 写作工具

### 主力：Obsidian

- **优点**：本地存储、双向链接、插件丰富
- **用途**：知识管理、文章草稿、灵感收集
- **技巧**：使用模板快速开始写作

### 发布：自定义网站

- 基于 Next.js 构建，完全可控
- Markdown 文件存储，方便管理

### 辅助工具

- **Grammarly**：英文语法检查
- **Hemingway Editor**：可读性分析
- **DeepL**：翻译辅助

## 设计工具

### 主力：Figma

- **用途**：UI 设计、图形设计、原型制作
- **优点**：协作方便、组件系统强大

### 图片处理

- **Photoshop**：复杂修图
- **ImageOptim**：图片压缩
- **Squoosh**：在线压缩

## 开发工具

### 编辑器：VS Code / Cursor

- 插件：Prettier, ESLint, GitLens, Tailwind CSS IntelliSense

### 版本控制：Git + GitHub

- 所有项目都用 Git 管理
- GitHub Actions 自动化部署

### 部署：Vercel

- 前端项目一键部署
- 自动 HTTPS、CDN、预览环境

## 效率工具

### 任务管理：Things 3

- 简洁优雅
- 支持项目、标签、截止日期

### 笔记：Apple Notes + Obsidian

- **Apple Notes**：快速记录、随手记
- **Obsidian**：深度思考、知识整理

### 阅读：Readwise Reader

- 稍后读管理
- 高亮同步
- 每日回顾

## 我的工具选择原则

1. **本地优先**：数据可控，离线可用
2. **简洁至上**：功能足够，不臃肿
3. **跨平台**：Mac + iOS + Web
4. **可导出**：避免被锁定
5. **性价比高**：免费或一次性付费优先

记住：最好的工具是你真正在用的工具。
