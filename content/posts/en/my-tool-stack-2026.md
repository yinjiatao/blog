## Why Tools Matter

Good tools can:

- Reduce creative friction
- Improve output quality
- Save time and energy
- Spark creative inspiration

But remember: tools are a means, not an end. Don't fall into the trap of tool obsession.

## Writing Tools

### Primary: Obsidian

- **Pros**: local storage, bi-directional links, rich plugins
- **Use**: knowledge management, article drafts, idea collection
- **Tip**: use templates to start writing quickly

### Publishing: Custom Website

- Built with Next.js, fully customizable
- Markdown file storage for easy management

### Auxiliary Tools

- **Grammarly**: English grammar checking
- **Hemingway Editor**: readability analysis
- **DeepL**: translation assistance

## Design Tools

### Primary: Figma

- **Use**: UI design, graphic design, prototyping
- **Pros**: great for collaboration, powerful component system

### Image Processing

- **Photoshop**: complex retouching
- **ImageOptim**: image compression
- **Squoosh**: online compression

## Development Tools

### Editor: VS Code / Cursor

- Plugins: Prettier, ESLint, GitLens, Tailwind CSS IntelliSense

### Version Control: Git + GitHub

- All projects managed with Git
- GitHub Actions for automated deployment

### Deployment: Vercel

- One-click deployment for frontend projects
- Automatic HTTPS, CDN, preview environments

## Productivity Tools

### Task Management: Things 3

- Clean and elegant
- Supports projects, tags, deadlines

### Notes: Apple Notes + Obsidian

- **Apple Notes**: quick capture, on-the-go notes
- **Obsidian**: deep thinking, knowledge organization

### Reading: Readwise Reader

- Read-later management
- Highlight sync
- Daily review

## My Tool Selection Principles

1. **Local first**: data control, offline availability
2. **Simplicity first**: sufficient functionality, not bloated
3. **Cross-platform**: Mac + iOS + Web
4. **Exportable**: avoid vendor lock-in
5. **Cost-effective**: prefer free or one-time payment

Remember: the best tool is the one you actually use.
